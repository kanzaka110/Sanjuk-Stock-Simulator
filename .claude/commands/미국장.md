## 3단계: 미국 보유종목 시세 + 시간외 + RIA 현황

```bash
cd /home/ohmil/Sanjuk-Stock-Simulator && source venv/bin/activate && python -c "
import yfinance as yf
from datetime import date
from core.market import _batch_quotes, _get_quote_realtime
from config.settings import HOLDINGS_GENERAL, PORTFOLIO, US_PORTFOLIO

# KIS API 우선 → yfinance 폴백으로 미국 종목 시세 조회
us_quotes = _batch_quotes(US_PORTFOLIO)
fx_quote = _get_quote_realtime('USDKRW=X')
fx = fx_quote.price if fx_quote else 1450.0
print(f'환율: {fx:,.2f}원')

# RIA 데드라인
deadline = date(2026, 5, 31)
d_day = (deadline - date.today()).days
print(f'RIA 100% 감면 데드라인: 2026-05-31 (D-{d_day})')
print()

# 미국 종목 상세
us_holdings = {tk: info for tk, info in HOLDINGS_GENERAL.items() if '.KS' not in tk}
total_pnl_krw = 0
ria_pnl_krw = 0

print('=== [일반] 해외주식 현황 (KIS → yfinance 폴백) ===')
for tk, info in us_holdings.items():
    try:
        q = us_quotes.get(tk)
        if not q: continue
        price = q.price
        shares = info['shares']
        avg = info['avg_cost_usd']
        pnl_usd = (price - avg) * shares
        pnl_pct = (price - avg) / avg * 100
        pnl_krw = pnl_usd * fx
        ria = info.get('ria_eligible', 0)
        ria_tag = f' [RIA {ria}주]' if ria > 0 else ''
        name = PORTFOLIO.get(tk, tk)

        # 시간외 가격 시도
        try:
            ext_info = yf.Ticker(tk).info
            pre = ext_info.get('preMarketPrice')
            post = ext_info.get('postMarketPrice')
            ext_price = post or pre
            ext_str = f' | 시간외: \${ext_price:,.2f}' if ext_price else ''
        except:
            ext_str = ''

        print(f'  {name:12s} {shares:>3d}주 | \${price:>8,.2f} ({q.pct:>+.2f}%){ext_str} | 평단:\${avg:>8,.2f} | 수익:\${pnl_usd:>+8,.0f} ({pnl_pct:>+.1f}%) ₩{pnl_krw/10000:>+6,.0f}만{ria_tag}')
        total_pnl_krw += pnl_krw
        if ria > 0:
            ria_pnl_krw += (price - avg) * ria * fx
    except Exception as e:
        print(f'  {tk}: ERROR {e}')

print(f'  ---')
print(f'  해외주식 합계 수익: ₩{total_pnl_krw/10000:>+,.0f}만')
print(f'  RIA 대상 이익: ₩{ria_pnl_krw/10000:,.0f}만 / 감면한도 ₩4,593만 (여유: ₩{(4593-ria_pnl_krw/10000):,.0f}만)')
"
```

## 4단계: 전체 포트폴리오 P&L + 서킷 브레이커

```bash
cd /home/ohmil/Sanjuk-Stock-Simulator && source venv/bin/activate && python -c "
from core.market import _batch_quotes, _get_quote_realtime
from config.settings import (
    HOLDINGS_GENERAL, HOLDINGS_ISA, HOLDINGS_IRP, HOLDINGS_PENSION,
    DEFAULT_CASH, ISA_CASH, IRP_CASH, IRP_DEFAULT_OPTION, PENSION_MMF,
    PORTFOLIO
)

# KIS API 우선 → yfinance 폴백으로 전 종목 일괄 조회
all_prices = _batch_quotes(PORTFOLIO)
fx_quote = _get_quote_realtime('USDKRW=X')
fx = fx_quote.price if fx_quote else 1450.0

all_accounts = {
    '일반': (HOLDINGS_GENERAL, DEFAULT_CASH),
    'ISA': (HOLDINGS_ISA, ISA_CASH),
    'IRP': (HOLDINGS_IRP, IRP_CASH + IRP_DEFAULT_OPTION),
    '연금': (HOLDINGS_PENSION, PENSION_MMF),
}

total_cost = total_val = 0
for acct, (holdings, cash) in all_accounts.items():
    acct_cost = acct_val = 0
    for tk, info in holdings.items():
        try:
            q = all_prices.get(tk)
            price = q.price if q else 0
            if not price: continue
            shares = info['shares']
            if 'avg_cost_usd' in info:
                cost = info['avg_cost_usd'] * shares * fx
                val = price * shares * fx
            else:
                avg = info.get('avg_cost_krw', info.get('avg_cost', 0))
                cost = avg * shares; val = price * shares
            acct_cost += cost; acct_val += val
        except: pass
    acct_val += cash
    pnl = (acct_val - acct_cost - cash) / acct_cost * 100 if acct_cost else 0
    print(f'[{acct:4s}] 평가:{acct_val/10000:>8,.0f}만 | 수익률:{pnl:>+.1f}%')
    total_cost += acct_cost; total_val += acct_val

total_pnl = (total_val - total_cost) / total_cost * 100 if total_cost else 0
print(f'[전체] 평가:{total_val/10000:>8,.0f}만 | 수익률:{total_pnl:>+.1f}%')
print(f'서킷브레이커: {\"🚨 발동\" if total_pnl < -5 else \"✅ 정상\"}')
"
```

### 출력 순서 (결론 먼저)

1. **최종 판단** — 맨 위. PASS 또는 구체적 액션. 계좌 태그 [일반][ISA][IRP][연금] 필수.
2. **뉴스 핵심 요약** — 3-Layer 교차 검증 결과. 복수 소스 확인 여부 표시.
3. **RIA 현황** — 대상 종목, 감면 이익, 데드라인 D-day, 매도 판단
4. **시장 환경** — 지수, VIX, 환율, 매크로 테이블 + 추세
5. **레짐** — 강세/약세/횡보/위기 + 전략 가이드
6. **포트폴리오 현황** — 계좌별 P&L 테이블 + 서킷 브레이커
7. **종목별 기술 분석** — RSI/MACD/BB/OBV/합류점수 + 시간외 가격. OBV 괴리 강조.
8. **리스크** — 상관관계, 변동성, 낙폭, 포지션 사이징
9. **백테스트** — 전략별 성과 비교, 레짐 기반 결과
10. **펀더멘털** — PER/EPS/매출성장/ROE/실적일정
11. **메모리** — 과거 추천 정확도
12. **매수 트리거 체크리스트**
13. **계좌별 액션** — 각 계좌 구체적 판단

### 매수 트리거 체크리스트 (수동 평가)

수집된 데이터를 기반으로 아래 트리거 충족 여부를 체크:

- [ ] VIX 30+ 급등 후 하락 전환
- [ ] RSI 30 이하 과매도 종목 존재
- [ ] 주요 지지선 터치 후 반등 확인
- [ ] OBV 상승 확인 (가격 + 거래량 동반)
- [ ] FOMC/CPI 등 이벤트 통과 후 방향성 확인
- [ ] 실적 발표 후 갭다운 과매도
- [ ] 52주 고점 대비 -20% 이상 할인

**트리거 2개 이상 충족 시에만 매수 고려. 그 외에는 PASS.**

### 계좌별 규칙 (반드시 준수)

- **[일반]**: **5/31까지 해외 신규매수 절대 금지** (RIA 한도 차감 방지). 매도만 가능.
  - RIA 대상: NVDA 46주, GOOGL 9주 (2025-12-23 이전 보유분)
  - 5/31까지 100% 감면 (한도 ₩4,593만)
  - RIA 매도 시점 판단: 이익 규모 + 시장 추세 + 세금 절감 효과 종합 고려
- **[ISA]**: 해외 주식 직접 매수 불가. 국내상장 ETF만.
- **[IRP]**: 리밸런싱만. 해외 직접 투자 불가.
- **[연금저축]**: 리밸런싱만. 해외 직접 투자 불가.

### 분석 필수 규칙

- 서킷브레이커 발동 시 → 전체 매매 중단, 분석만 제공
- OBV와 RSI가 불일치 → 저점 미확인, 매수 보류
- 백테스트에서 B&H 대비 초과 수익 없는 전략 → 단기매매 비추천
- 감성지표 vs 기술지표 괴리 → 주의 표시
- 확신 없으면 **"패스"가 정답**
- 신규 종목 추천은 명확한 근거 있을 때만 (없으면 "없음" 명시)
- RIA 매도 판단: D-30 이내 진입 시 본격 검토, 그 전에는 추세 관찰
