## 3단계: 전체 포트폴리오 P&L + RIA + 서킷 브레이커

```bash
cd /home/ohmil/Sanjuk-Stock-Simulator && source venv/bin/activate && python -c "
from datetime import date
from core.market import _batch_quotes, _get_quote_realtime
from config.settings import (
    HOLDINGS_GENERAL, HOLDINGS_ISA, HOLDINGS_IRP, HOLDINGS_PENSION,
    DEFAULT_CASH, ISA_CASH, IRP_CASH, IRP_DEFAULT_OPTION, PENSION_MMF,
    PORTFOLIO, PRICE_TARGETS
)

all_prices = _batch_quotes(PORTFOLIO)
fx_quote = _get_quote_realtime('USDKRW=X')
fx = fx_quote.price if fx_quote else 1450.0
print(f'환율: {fx:,.2f}원')

# RIA
deadline = date(2026, 5, 31)
d_day = (deadline - date.today()).days
print(f'RIA 100% 감면: D-{d_day} (2026-05-31)')

all_accounts = {
    '일반': (HOLDINGS_GENERAL, DEFAULT_CASH),
    'ISA': (HOLDINGS_ISA, ISA_CASH),
    'IRP': (HOLDINGS_IRP, IRP_CASH + IRP_DEFAULT_OPTION),
    '연금': (HOLDINGS_PENSION, PENSION_MMF),
}

total_cost = total_val = 0
ria_pnl = 0
for acct, (holdings, cash) in all_accounts.items():
    acct_cost = acct_val = 0
    for tk, info in holdings.items():
        try:
            q = all_prices.get(tk)
            price = q.price if q else 0
            if not price: continue
            shares = info['shares']
            if 'avg_cost_usd' in info:
                cost = info['avg_cost_usd'] * shares * fx
                val = price * shares * fx
                ria = info.get('ria_eligible', 0)
                if ria > 0:
                    ria_pnl += (price - info['avg_cost_usd']) * ria * fx
            else:
                avg = info.get('avg_cost_krw', info.get('avg_cost', 0))
                cost = avg * shares; val = price * shares
            pnl = (val - cost) / cost * 100 if cost else 0
            name = PORTFOLIO.get(tk, tk)
            chg_str = f'({q.pct:>+.2f}%)' if q and q.pct else ''
            print(f'[{acct:4s}] {name:16s} | {shares:>4d}주 | 현재:{price:>10,.0f} {chg_str} | {pnl:>+.1f}%')
            acct_cost += cost; acct_val += val
        except Exception as e:
            print(f'[{acct:4s}] {tk}: ERROR {e}')
    acct_val += cash
    pnl = (acct_val - acct_cost - cash) / acct_cost * 100 if acct_cost else 0
    print(f'[{acct:4s}] 합계: 평가 {acct_val/10000:,.0f}만 | 예수금 {cash/10000:,.0f}만 | 수익률 {pnl:>+.1f}%')
    total_cost += acct_cost; total_val += acct_val

total_pnl = (total_val - total_cost) / total_cost * 100 if total_cost else 0
print(f'=== 전체: 평가 {total_val/10000:,.0f}만 | 수익률 {total_pnl:>+.1f}% ===')
print(f'RIA 대상 이익: ₩{ria_pnl/10000:,.0f}만 / 한도 ₩4,593만')
print(f'서킷브레이커: {\"🚨 발동 (매매 중단)\" if total_pnl < -5 else \"✅ 정상\"}')

print()
print('=== 목표가 모니터링 ===')
for tk, info in PRICE_TARGETS.items():
    q = all_prices.get(tk)
    if q:
        gap = (q.price - info['target']) / info['target'] * 100
        status = '🎯 도달!' if q.price <= info['target'] else f'현재 {q.price:,.0f} (목표까지 {gap:>+.1f}%)'
        print(f'{info[\"name\"]:16s} | 목표: {info[\"target\"]:>10,.0f} | {status}')
"
```

## 분석 및 출력 지침

위 데이터를 모두 수집한 후, 아래 순서와 규칙에 따라 브리핑을 작성합니다.

### 출력 순서 (결론 먼저)

1. **최종 판단** — PASS 또는 구체적 액션. 계좌 태그 [일반][ISA][IRP][연금] 필수.
2. **뉴스 핵심 요약** — 3-Layer 교차 검증 결과. 한국/미국 분리.
3. **RIA 현황** — 대상 종목, 감면 이익, D-day
4. **시장 환경** — 한국+미국 지수, VIX, 환율, 매크로
5. **레짐** — 강세/약세/횡보/위기 + 전략 가이드
6. **포트폴리오 현황** — 계좌별 P&L 테이블 + 서킷 브레이커
7. **종목별 기술 분석** — 한국+미국 전 종목. OBV 괴리 강조.
8. **수급** — KRX 기관/외국인 매매 동향
9. **리스크** — 상관관계, 변동성, 낙폭
10. **백테스트** — 전략별 성과
11. **펀더멘털** — PER/EPS/실적일정
12. **메모리** — 과거 추천 정확도
13. **매수 트리거 체크리스트**
14. **계좌별 액션** — 각 계좌 구체적 판단

### 매수 트리거 체크리스트

- [ ] VIX 30+ 급등 후 하락 전환
- [ ] RSI 30 이하 과매도 종목 존재
- [ ] 주요 지지선 터치 후 반등 확인
- [ ] 기관/외국인 순매수 전환
- [ ] OBV 상승 확인 (가격 + 거래량 동반)
- [ ] FOMC/CPI 등 이벤트 통과 후 방향성 확인
- [ ] 실적 발표 후 갭다운 과매도

**트리거 2개 이상 충족 시에만 매수 고려. 그 외에는 PASS.**

### 계좌별 규칙

- **[일반]**: 5/31까지 해외 신규매수 금지 (RIA). 국내 매매 자유.
- **[ISA]**: 국내주식 + 국내상장 ETF만. 예수금 확인.
- **[IRP]**: 리밸런싱만.
- **[연금저축]**: 리밸런싱만.
- **RIA 대상**: NVDA 46주, GOOGL 9주. 5/31 100% 감면.

### 분석 필수 규칙

- 서킷브레이커 발동 시 → 전체 매매 중단
- OBV vs RSI 불일치 → 매수 보류
- B&H 대비 초과 수익 없는 전략 → 단기매매 비추천
- 확신 없으면 **"패스"가 정답**
- 신규 종목 추천은 명확한 근거 있을 때만
