> ⚠️ **시각 표기 필수 규칙**: 서버는 UTC — 브리핑 제목/본문의 시각은 반드시 KST로 변환할 것.
> Python: `from config.settings import KST; from datetime import datetime; datetime.now(KST).strftime("%Y-%m-%d %H:%M")`
> Bash 확인: `TZ=Asia/Seoul date`. naive `datetime.now()` 사용 금지 (9시간 오차).


# 미국장 프리브리핑 (오늘 밤 미국장 대비)

/미국장 브리핑과 동일한 분석을 수행하되, **오늘 밤 미국장 지정가 예약 주문**에 초점을 맞춥니다.

**핵심 차이점:**
- 모든 매수 추천에 **구체적 지정가** 필수 (범위가 아닌 조건부 가격)
- **프리마켓/선물 시나리오별 대응** (선물 상승/보합/하락)
- **"오늘 밤 지정가 주문 요약"** 테이블 포함 — HTS 바로 입력 가능
- **주문 넣고 자도 되는지** 판단 포함

## 출력 특화 규칙 (미국장 프리브리핑)

### 결론 포맷

브리핑 최상단에 **오늘 밤 지정가 주문 요약 테이블**을 배치:

```
🌙 오늘 밤 지정가 주문 요약 (HTS 입력용)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

종목명    | 계좌   | 지정가       | 수량  | 유효기간  | 조건
MU       | [일반]  | $98.00 이하  | 5주   | DAY      | 프리마켓 약세 시 $95.00
LMT      | [일반]  | $480.00     | 1주   | GTC      | —

📊 선물/프리마켓 시나리오
• 선물 상승 +1% 이상: 추격 매수 자제, 눌림목 대기 ($XX.XX)
• 선물 보합: 위 지정가 유지
• 선물 하락 -1% 이상: 하향 지정가 ($XX.XX), 추가 매수 기회

😴 주문 후 취침 가능 여부: [가능/모니터링 필요] — 근거
```

### 분석 필수 규칙 (기본 규칙 + 프리브리핑 추가)

- 모든 매수 추천에 **"$XX.XX 이하" 형식 지정가** 필수
- 주문 유효 기간 명시 (DAY / GTC)
- B안 전략, RIA 규칙은 /미국장과 동일 적용
- **주문 넣고 자도 되는지** 명확히 판단 (변동성 높으면 모니터링 권고)

### Notion + 텔레그램 전송

/미국장과 동일한 Notion 저장 → 텔레그램 전송 수행. 텔레그램 메시지 헤더에 🌙 표시.
