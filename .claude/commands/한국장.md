### 0-2. WebSearch 심층 검증 (Claude가 직접 수행)

위 뉴스에서 발견된 핵심 이슈를 WebSearch로 교차 검증합니다:

1. **보유 종목 핵심 뉴스** — 삼성전자, 한화에어로스페이스 관련 주요 뉴스를 검색하여 RSS/네이버 뉴스와 대조
2. **시장 이벤트** — FOMC/CPI/실적 발표 일정, 지정학 이슈 확인
3. **증권사 리포트** — 보유 종목 목표가 변경, 투자의견 변경 검색
4. **매크로** — 환율 전망, 금리 결정, 유가 동향 심층 확인

### 교차 검증 규칙

뉴스 분석 시 아래 기준으로 신뢰도를 판단:
- **복수 소스 확인** (RSS + 네이버 + WebSearch) → ✅ 고신뢰
- **단일 소스만** → ⚠️ 사실 여부 명시하고 주의 표시
- **소스 간 상충** → 🚨 양쪽 관점 모두 제시하고 경고
- **yfinance 영문 뉴스 vs 한국 뉴스 괴리** → 시차/관점 차이 구분

## 3단계: 포트폴리오 P&L + 서킷 브레이커

```bash
cd /home/ohmil/Sanjuk-Stock-Simulator && source venv/bin/activate && python -c "
from core.market import _batch_quotes, _get_quote_realtime
from config.settings import (
    HOLDINGS_GENERAL, HOLDINGS_ISA, HOLDINGS_IRP, HOLDINGS_PENSION,
    DEFAULT_CASH, ISA_CASH, IRP_CASH, IRP_DEFAULT_OPTION, PENSION_MMF,
    PORTFOLIO, PRICE_TARGETS
)

# 전 종목 시세를 KIS API 우선 → yfinance 폴백으로 일괄 조회
all_prices = _batch_quotes(PORTFOLIO)
fx_quote = _get_quote_realtime('USDKRW=X')
fx = fx_quote.price if fx_quote else 1450.0
print(f'환율: {fx:,.2f}원')

all_accounts = {
    '일반': (HOLDINGS_GENERAL, DEFAULT_CASH),
    'ISA': (HOLDINGS_ISA, ISA_CASH),
    'IRP': (HOLDINGS_IRP, IRP_CASH + IRP_DEFAULT_OPTION),
    '연금': (HOLDINGS_PENSION, PENSION_MMF),
}

total_cost = total_val = 0
for acct, (holdings, cash) in all_accounts.items():
    acct_cost = acct_val = 0
    for tk, info in holdings.items():
        try:
            q = all_prices.get(tk)
            price = q.price if q else 0
            if not price: continue
            shares = info['shares']
            if 'avg_cost_usd' in info:
                cost = info['avg_cost_usd'] * shares * fx
                val = price * shares * fx
            else:
                avg = info.get('avg_cost_krw', info.get('avg_cost', 0))
                cost = avg * shares
                val = price * shares
            pnl = (val - cost) / cost * 100 if cost else 0
            name = PORTFOLIO.get(tk, tk)
            chg_str = f'({q.pct:>+.2f}%)' if q and q.pct else ''
            print(f'[{acct:4s}] {name:16s} | {shares:>4d}주 | 현재:{price:>10,.0f} {chg_str} | {pnl:>+.1f}%')
            acct_cost += cost; acct_val += val
        except Exception as e:
            print(f'[{acct:4s}] {tk}: ERROR {e}')
    acct_val += cash
    pnl = (acct_val - acct_cost - cash) / acct_cost * 100 if acct_cost else 0
    print(f'[{acct:4s}] 합계: 평가 {acct_val/10000:,.0f}만 | 예수금 {cash/10000:,.0f}만 | 수익률 {pnl:>+.1f}%')
    total_cost += acct_cost; total_val += acct_val

total_pnl = (total_val - total_cost) / total_cost * 100 if total_cost else 0
print(f'=== 전체: 평가 {total_val/10000:,.0f}만 | 수익률 {total_pnl:>+.1f}% ===')
print(f'서킷브레이커: {\"🚨 발동 (매매 중단)\" if total_pnl < -5 else \"✅ 정상\"}')

# 2차 매수 목표가 체크
print()
print('=== 목표가 모니터링 ===')
for tk, info in PRICE_TARGETS.items():
    q = all_prices.get(tk)
    if q:
        gap = (q.price - info['target']) / info['target'] * 100
        status = '🎯 도달!' if q.price <= info['target'] else f'현재 {q.price:,.0f} (목표까지 {gap:>+.1f}%)'
        print(f'{info[\"name\"]:16s} | 목표: {info[\"target\"]:>10,.0f} | {status} | {info[\"reason\"]}')
"
```

## 분석 및 출력 지침

위 9단계 데이터를 모두 수집한 후, 아래 순서와 규칙에 따라 브리핑을 작성합니다.

### 출력 순서 (결론 먼저)

1. **최종 판단** — 맨 위. PASS 또는 구체적 액션. 계좌 태그 [일반][ISA][IRP][연금] 필수.
2. **뉴스 핵심 요약** — 3-Layer 교차 검증 결과. 복수 소스 확인 여부 표시.
3. **시장 환경** — 지수, VIX, 환율, 매크로 테이블 + 추세
4. **레짐** — 강세/약세/횡보/위기 + 전략 가이드
5. **포트폴리오 현황** — 계좌별 P&L 테이블 + 서킷 브레이커
6. **종목별 기술 분석** — RSI/MACD/BB/OBV/합류점수. OBV 괴리 강조.
7. **수급** — KRX 기관/외국인 매매 동향
8. **리스크** — 상관관계, 변동성, 낙폭, 포지션 사이징
9. **백테스트** — 전략별 성과 비교, 레짐 기반 결과
10. **펀더멘털** — PER/EPS/실적일정
11. **메모리** — 과거 추천 정확도
12. **매수 트리거 체크리스트**
13. **계좌별 액션** — 각 계좌 구체적 판단

### 매수 트리거 체크리스트 (수동 평가)

수집된 데이터를 기반으로 아래 트리거 충족 여부를 체크:

- [ ] VIX 30+ 급등 후 하락 전환
- [ ] RSI 30 이하 과매도 종목 존재
- [ ] 주요 지지선 터치 후 반등 확인
- [ ] 기관/외국인 순매수 전환
- [ ] OBV 상승 확인 (가격 + 거래량 동반)
- [ ] FOMC/CPI 등 이벤트 통과 후 방향성 확인
- [ ] 실적 발표 후 갭다운 과매도

**트리거 2개 이상 충족 시에만 매수 고려. 그 외에는 PASS.**

### 계좌별 규칙 (반드시 준수)

- **[일반]**: 5/31까지 해외 신규매수 금지 (RIA 한도 차감 방지). 국내 매매는 자유.
- **[ISA]**: 국내주식 + 국내상장 ETF만 매수 가능. 예수금 확인.
- **[IRP]**: 리밸런싱만 (매수와 매도 동시). 단독 매수 불가.
- **[연금저축]**: 리밸런싱만. 단독 매수 불가.
- **RIA 대상**: NVDA 46주, GOOGL 9주 (일반계좌). 5/31 100% 감면 기한.

### 분석 필수 규칙

- 서킷브레이커 발동 시 → 전체 매매 중단, 분석만 제공
- OBV와 RSI가 불일치 → 저점 미확인, 매수 보류
- 백테스트에서 B&H 대비 초과 수익 없는 전략 → 단기매매 비추천
- 감성지표 vs 기술지표 괴리 → 주의 표시
- 확신 없으면 **"패스"가 정답**
- 신규 종목 추천은 명확한 근거 있을 때만 (없으면 "없음" 명시)
